Installation on GNU Linux:

Precompiled packages

Please choose the appropriate package for your Linux distribution or Windows, see Home:Download

Build from sources

Requirements

  • Fedora 20, 21: 

sudo yum install gcc-c++ intltool gtkmm30-devel poppler-glib-devel

  • Ubuntu 14.04, 14.10: 

sudo apt install gcc intltool libgtkmm-3.0-dev libpoppler-glib-dev

  • openSuse:

sudo zypper install gcc-c++ intltool gtkmm3-devel libpoppler-glib-devel

(Alternatively, use su -c "..." instead of sudo ...)

Compile and install

wget https://github.com/dertoki/pdfMelange/archive/0.4.2.tar.gz -O pdfMelange-0.4.2.tar.gz
tar xvzf pdfMelange-0.4.2.tar.gz
cd pdfMelange-0.4.2
./configure
make
sudo make install

Uninstall

cd pdfMelange-0.4.2
sudo make uninstall
