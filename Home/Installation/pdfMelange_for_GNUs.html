Installation on GNU Linux:

Precompiled packagues

Please choose the appropriate package for your Linux distribution or Windows, see Home:Download

Build from sources

Requirements

  • Fedora 20:
sudo yum install gcc-c++ intltool gtkmm30-devel poppler-glib-devel python2-devel
  • Ubuntu 14.04:
sudo apt install gcc intltool libgtkmm-3.0-dev libpoppler-glib-dev libpython2.7-dev
  • openSuse:
sudo zypper install gcc-c++ intltool gtkmm3-devel libpoppler-glib-devel python-devel

(Alternatively, use su -c "..." instead of sudo ...)

Compile and install

wget ...
tar xvzf pdfMelange-0.4.1.tar.gz
cd pdfMelange-0.4.1
./configure
make
sudo make install

Uninstall

cd pdfMelange-0.4.1
sudo make uninstall
