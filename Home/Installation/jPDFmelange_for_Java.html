jPDFmelange for Java

Version 0.1 to 0.3 was developed with java in 2008.
The java version is no longer maintained.

Installation

As a Java program, jPDFmelange should work nearly platform independent.

The installation of a Java Runtime Environment is required.

This should work on Linux, Windows and MacOS:

  1. Download jPDFmelange-0.3.4.zip.
  2. Extract the zip archiv.
  3. Start a command line.
  4. Change the current directory to jPDFmelange.
  5. Execute 

java -jar jPDFmelange-0.3.4.jar

Uninstall

Simply remove the program folder jPDFmelange from your hard drive.
