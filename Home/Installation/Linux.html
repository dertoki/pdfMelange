Installation on GNU Linux:

Precompiled packages


Please choose the appropriate package for your Linux distribution, see Home:Download

Build from sources


Requirements


  • Fedora: 

sudo dnf install gcc-c++ intltool sysconftool gtkmm30-devel poppler-glib-devel

  • Ubuntu: 

sudo apt install gcc intltool libgtkmm-3.0-dev libpoppler-glib-dev libpoppler-private-dev

  • openSuse:

sudo zypper install gcc-c++ intltool gtkmm3-devel libpoppler-glib-devel

(Alternatively, use su -c "..." instead of sudo ...)


Compile and install

wget https://github.com/dertoki/pdfMelange/archive/v0.4.4.tar.gz -O pdfMelange-0.4.4.tar.gz
tar xvzf pdfMelange-0.4.4.tar.gz
cd pdfMelange-0.4.4
./configure
make
sudo make install

Uninstall

cd pdfMelange-0.4.4
sudo make uninstall
