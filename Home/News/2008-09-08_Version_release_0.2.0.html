2008-09-08 Version release 0.2.0

  • Drag and Drop is enabled with external file manager, tested with Windows 98/XP, KDE 3.5 and Gnome 2.22.
  • DnD is also enabled for internal lists.
  • You can modify the page order or throw pages to the buffer list with DnD.
  • Key bindings (delete key) are enabled for the lists.
