Development Resources

Summary


Version 0.4 has been developed since 2013 with C++ and Gtkmm as a Gnome application.

  • To build the C++ version from sources, you simply need a linux box, see +build from C++ sources for details.
  • It is also available for Windows OSes. To build pdfMelange for Windows OSes see +build with minGW .

Version 0.1 to 0.3 was developed with java in 2008

  • The java version is no longer maintained.
  • To build the java version from sources, you need a java jdk version 6 or higher and some other dependencies, see +build from java sources for details.
