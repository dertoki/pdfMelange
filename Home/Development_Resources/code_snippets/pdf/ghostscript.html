ghostscript

Created Montag 06 Oktober 2014

Ghostscript seems to be a good opportunity for merging pages.

Here an example that merges the PDF files "123.pdf" page 1, "abc.pdf" page 2 and "123.pdf" page 4 into "output.pdf":
gs -sDEVICE=pdfwrite -dNOPAUSE -dQUIET -dBATCH \
-sOutputFile=output.pdf \
-dFirstPage=1 -dLastPage=1 123.pdf \
-dFirstPage=2 -dLastPage=2 abc.pdf \
-dFirstPage=4 -dLastPage=4 123.pdf

There is a C-API for ghostscript:
