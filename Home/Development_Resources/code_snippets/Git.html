The following snippets are my mnemonics.

Push files to stage

$ git add --all .

Commit staged files

$ git commit

Commit and push staged files

$ git commit -a -m 'comment ...'

Check for modified files that are not staged

$ git status

Show history of commits

$ git log
$ git log --pretty=oneline

Unstage file

$ git rm --cached <file>

Adding a remote

$ git remote add origin https://github.com/<gituser>/<project>

Push Pages

$ git push origin master:gh-pages

Push Pages (force the push of the gh-pages branch to the remote gh-pages branch at origin)

$ git push --force origin master:gh-pages

Create a branch

$ git branch testing

Change to a branch

$ git checkout testing

Change to a branch master

$ git checkout master

Merge changes in master to testing

$ git commit -a -m 'comment ...'
$ git checkout testing
$ git merge master

Merge changes in testing to master

$ git commit -a -m 'comment ...'
$ git checkout master
$ git merge testing

Create a tag

$ git tag -a 0.4.1 -m 'v0.4.1'

Checkout a special file

$ git checkout -- <file>
