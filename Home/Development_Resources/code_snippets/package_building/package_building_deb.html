Workflow:

with dpkg-buildpackage:
tar xvf pdfmelange-0.4.2.tar.gz
cd pdfmelange-0.4.2
dh_make -f ../pdfmelange-0.4.2.tar.gz 
dpkg-buildpackage -S -us -uc 
dpkg-buildpackage -us -uc
