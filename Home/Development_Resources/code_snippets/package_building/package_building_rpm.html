package building rpm

Links

  • Intro: http://fedoraproject.org/wiki/How_to_create_a_GNU_Hello_RPM_package
  • Howto: http://fedoraproject.org/wiki/How_to_create_an_RPM_package

Workflow:

  1. Set up a RPM build area in ~/rpmbuild.

rpmdev-setuptree

  1. Copy source .tar.gz into ~/rpmbuild/SOURCES
  2. Create a .spec file in ~/rpmbuild/SPECS

cd ~/rpmbuild/SPECS
rpmdev-newspec

  1. Edit the .spec file according to http://fedoraproject.org/wiki/How_to_create_an_RPM_package#Spec_file_pieces_explained
  2. Build the package

rpmbuild -ba hello.spec
